This project is an interactive Sales Dashboard built using Python, SQL, Pandas, and Power BI to analyze and visualize sales performance data. The dashboard provides meaningful business insights by transforming raw sales data into clear, actionable visual reports that help stakeholders monitor revenue, trends, and key performance indicators (KPIs).
The goal of this project is to demonstrate end-to-end data analytics skills, including data extraction, cleaning, analysis, and visualization.

Tech Stack:-
Python – Data processing and analysis
SQL – Data extraction and querying
Pandas – Data cleaning, transformation, and aggregation
Power BI – Data modeling, visualization, and dashboard creation

Workflow
Data Collection :-Sales data extracted from a relational database using SQL queries.
Data Cleaning & Transformation :-Cleaned missing values, corrected data types, and performed aggregations using Python and Pandas.
Data Analysis:-Calculated KPIs such as total sales, revenue trends, average order value, and product performance.
Dashboard Creation:-Imported processed data into Power BI.
